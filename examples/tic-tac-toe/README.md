# Tic-Tac-Toe powered by @bame

This is a basic Tic-Tac-Toe project designed to demonstrate how the **@bame** engine works. It serves as a reference for utilizing the engine to build powerful, multiplayer online turn-based games.

## Features

- **Turn-based Logic**: Powered by @bame's core mechanics.
- **Multiplayer Ready**: Built with multiplayer connectivity in mind.
- **Example Implementation**: Clean and simple code to help you get started with the @bame ecosystem.

## Getting Started

First, ensure you have dependencies installed in the root workspace. Then, run the development server from this directory or via the root:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Learn More

To learn more about the @bame engine, check the documentation in the `packages/core` directory.
